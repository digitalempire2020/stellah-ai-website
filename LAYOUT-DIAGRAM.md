## Responsive Breakpoint Behavior

```
320px (Mobile S)
├─ Single column
├─ Stacked cards
├─ Reduced padding (p-6)
├─ Smaller text (base sizes)
└─ Simplified animations

640px (Mobile L / Tablet)
├─ Increased padding (p-8)
├─ Larger headings (sm: classes)
└─ More breathing room

768px (Tablet)
├─ Some side-by-side layouts begin (md: classes)
├─ Comparison cards still stack
└─ Agent cards in grid

1024px (Desktop)
├─ Full side-by-side layouts (lg: classes)
├─ Comparison cards horizontal
├─ Maximum padding (p-12)
└─ All animations enabled

1280px+ (Large Desktop)
├─ Content max-width constrains (max-w-7xl)
├─ Centered with margins
└─ Optimal line lengths
```
